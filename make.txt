8.Cilj predmeta
Unapređenje postojećih znanja studenata iz oblasti XML-a i pružanje novih znanja, tehnika i veština rada
sa naprednim tehnologijama i aplikacijama zasnovanim na XML-u.

9.Ishod predmeta
Student će biti sposoban da kreira, analizira, obrađuje i transformiše XML dokumenta upotrebom
naprednih XML tehnologija. Osim toga, biće u mogućnosti da primeni stečena znanja u različitim
implemtacionim okvirima na svim slojevima arhitekture aplikacija (sloju baze podataka, sloju logike
aplikacija i prezentacionom sloju).

10.Sadržaj predmeta

11.Teorijska nastava:
Osnove XML-a (DTD, XML Schema). Povezivanje i navigacija (XLink, XPointer, XPath). Transformacije
XML dokumenata (XSLT). Prikaz XML sadržaja na WEB-u (XHTML, CSS). Formatiranje XML podataka
(XSL-FO). Kreiranje ekranskih formi za obradu XML dokumenata (XForms). Alati za kreiranje i obradu
XML-a. Parsiranje XML dokumenata (DOM i SAX). Obrada XML-a u Javi. XML u aplikacionim
okruženjima (Coocun, JSF, MyBatis, Hibernate, EJB). Upotreba XML za konfigurisanje i transport
podataka. Generisanje XML dokumentima iz objektno-relacionih baza podatka. Skladištenje i obrada
XML-a u bazama podatka. Semantički XML modeli podataka (RDFS, OWL). Priprema za ispit.

12.Praktična nastava:
Osnovni pojmovi XML dokumenata: Izrada jednostavnih primera. Osnovni pojmovi XML dokumenata:
Izrada kompleksnih primera. Transformacije XML dokumenata upotrebom XSLT-a: Izrada jednostavnih
primera. Transformacije XML dokumenata upotrebom XSLT-a: Izrada kompleksnih primera. Prikaz XML
sadržaja na WEB-u (XHTML, CSS). Formatiranje XML podataka upotrebom XSL-FO funkcija i tipova
podataka. Upotreba XForms-a za kreiranje ekranskih formi. Parsiranje XML dokumenata upotrebom DOM
i SAX: Izrada jednostavnih primera. Parsiranje XML dokumenata upotrebom DOM i SAX: Izrada
kompleksnih primera. Prikaz i upotreba XML u aplikacionim okruženjima. Upotreba SQL-a za
generisanje XML-a iz objektno-relacionih baza podataka. Skladištenje i obrada XML-a u bazama podataka.
XML i JavaScript. Upotreba web-servisa za manipulaciju XML –om. Priprema za ispit.
